# 🛒 GroceryMart

A modern, interactive grocery web application built with Django and custom front-end design.

GroceryMart is designed with a Gen-Z inspired UI featuring glassmorphism, neon accents, animated components, and interactive shopping features.

---

## 🚀 Features

### 🏠 Home Page
- Modern hero section
- Interactive category cards
- Animated UI elements
- Neon + glass design theme
